# 隐私说明

人话改写器是一个纯前端工具。

## 本地处理

以下内容只在你的浏览器里处理：

- AI 味检测
- 风格和强度选择
- 输入字数统计
- 输出滚动和复制状态

## 会发送到 OpenRouter 的内容

点击「改成人话」后，工具会把以下内容发送到 OpenRouter：

- 你输入的原文
- 当前选择的改写规则
- 当前选择的模型

OpenRouter 会再把请求转发给你选择的模型提供方。模型提供方的处理方式以它们自己的隐私政策为准。

## API key 保存方式

OpenRouter API key 保存在当前浏览器的 `localStorage` 里。

`localStorage` 是浏览器提供的本地存储空间，数据保存在你自己的设备上。它不会自动同步到这个项目的作者，也不会出现在 GitHub 仓库里。

你可以在页面里点击「清除」删除 key。

## 安全提醒

不要把 API key 提交到 GitHub，不要贴到 issue，不要放进截图里。API key 泄露后，别人可能用你的账号额度调用模型。
